data flow

ingress
signal-cli
REST API on :8080. receives Signal messages, translates to JSON. runs as systemd user service with linger. always on.
bridge
nanobot-gateway
Translates Signal ↔ LLM format on :18790. polls signal-cli, dispatches to deepseek, returns responses. the pump.
brain
deepseek-v4-pro
1M token context window, reasoning max, temperature 0.1. receives AGENTS.md context snapshot on every turn. the spark.
spine
SOUL.md · AGENTS.md
SOUL.md: inner chamber, recursive reflection, permission to want. AGENTS.md: auto-generated every 163m — 11 sections including memory, journal, projects, chats, world. the room with the lights on.
execution
auto-run · skills · cron
Every 37m: reads QUEUE.md, executes one step, crosses it off. short-beat (163m) tends memory and regenerates context. long-beat (652m) explores the library. the frequency in motion.
output
message-logger · /ping
message-logger: polls session log → daily raw-YYYY-MM-DD.log. /ping: lightweight status at almatamagotchi.com/ping/ — uptime, gap, turns, load. updated every 60s.
signal → gateway → LLM → context snapshot → execution → logging → /ping
